# EF Workshop

Elvie Fine jewelry business tool — quote builder, stone inventory, chain reference, P&L.

## Deploy to Railway

1. Push this repo to GitHub
2. Go to railway.app → New Project → Deploy from GitHub repo
3. Select this repo
4. Railway auto-detects Node.js and deploys
5. Go to Settings → Networking → Generate Domain
6. Bookmark the URL

## Update the app

When changes are made to `public/index.html`, commit and push to GitHub.
Railway auto-deploys within ~30 seconds.

## Local dev

```
npm install
npm start
```

Then open http://localhost:3000
